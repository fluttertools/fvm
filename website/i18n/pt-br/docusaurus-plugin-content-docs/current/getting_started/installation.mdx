Existem algumas maneiras de instalar o FVM e estamos trabalhando em outras maneiras de torná-lo ainda mais simples.

## Standalone (Recomendado) {#standalone-recommended}

Você pode baixar os pacotes autônomos do repositório GitHub [aqui](https://github.com/leoafarias/fvm/releases).

Se você usa o [gerenciador de pacotes Homebrew](https://brew.sh) para Mac OS X, você pode instalar o FVM executando

brew install fvm

brew uninstall fvm

## Pub package {#pub-package}

Você também pode instalar o FVM como um [pacote pub](https://pub.dev/packages/fvm).

No entanto, se você planeja usar o FVM para gerenciar sua [instalação global do Flutter](/docs/guides/global_version), recomendamos instalar como autônomo.

```bash
dart pub global activate fvm
```
